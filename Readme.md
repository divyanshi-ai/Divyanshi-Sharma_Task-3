## 🚀 How to Run & View

### Method 1: View the Live Deployment (Recommended)
We can view the fully functioning, hosted version of this project instantly by clicking the link after deploying the project on github/vercel/netlify.

If we want to view or test the source code locally:
1. Download or clone this repository to your machine.
2. Navigate to the project's root folder.
3. Double-click the **`index.html`** file to open it instantly in any web browser.
*(Alternatively, right-click `index.html` in VS Code and select **"Open with Live Server"**).*

 ## 🚀 Features & Technical Implementation

This project implements various CSS selection techniques to style specific elements exactly as per the layout requirements:

* **Heading Typography:** Managed specific header hierarchies (`h1`, `h2`, `h4`) using a uniform `.head` class.
* **ID Overrides:** Leveraged a `#bglime` ID to cleanly override background styling on specific header elements.
* **Paragraph Layouts:** Utilized global `p` element selectors alongside `#para` target IDs for fine-tuned paragraph customization.
* **Advanced List & Link Selection:** Styled anchor (`a`) tags and selective list items (`li:nth-child()`) cleanly from the stylesheet without injecting any inline CSS or cluttering the markup with extra classes/IDs.
